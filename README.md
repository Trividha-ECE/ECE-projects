## Project 3: 5V to 10V Boost Converter using MT3608

### Description:

Designed a DC-DC boost converter PCB to step up 5V input to ~10V output using MT3608 IC.

### Software Used:

* KiCad

### Components Used:

* MT3608 IC
* Inductor
* Diode
* Capacitors
* Resistors

### Features:

* Compact PCB design
* Proper switching layout
* Stable output using feedback resistors

### Output:

* Input: 5V
* Output: ~10V
